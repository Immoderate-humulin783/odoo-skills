# Repository Rules

Skills live directly under `skills/`. Do not add bucket folders such as `engineering/`.

Every skill must have a `SKILL.md` with frontmatter containing `name` and `description`.

Every public skill must be listed in:

- `README.md`
- `skills/README.md`
- `.claude-plugin/plugin.json`

When changing skill routing, keep top-level docs and plugin entries in sync.
